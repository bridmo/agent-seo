## Comment remplir ce fichier

### Si vous avez des transcripts de calls (ideal)

```
Voici [N] transcripts de calls commerciaux/support.
Analyse-les et remplis chaque section de ce template icp-bible.md.
Pour chaque pain point, extrais les citations exactes.
Classe par frequence (mentionne dans X% des calls).
```

### Si vous avez un CRM (HubSpot, Salesforce...)

```
Extrais les donnees de nos [N] derniers clients :
- Poste, secteur, taille entreprise
- Source d'acquisition
- Temps avant conversion
- Notes des commerciaux
Puis remplis les sections demographiques et parcours d'achat.
```

### Si vous n'avez rien

```
Je vais te decrire mes clients de memoire.
Pose-moi des questions pour chaque section du template.
On va construire l'ICP bible ensemble.
```

> L'ICP bible est un document vivant. Revenez l'enrichir a chaque nouveau call, chaque nouvelle enquete, chaque nouveau retour client. Plus il est riche, meilleurs seront les articles.
